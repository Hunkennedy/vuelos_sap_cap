# Run the project


* Install Nodejs and NPM
* Install CDS
> npm i -g @sap/cds-dk
* Clone the repository
> git clone https://github.com/Hunkennedy/vuelos_sap_cap
* Install dependencies
> npm install
* Fill local DB
> npm run sqlite
* Run
> npm run dev
